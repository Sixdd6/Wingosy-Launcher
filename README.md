# Wingosy-Launcher-Sixdd
> A (fork of a) game launcher and cloud save sync client for RomM on Windows. Browse your library, launch games, and keep saves backed up automatically (sometimes).

## Setup
> Usage requires a working RomM instance on your own hardware. More information on this can be found (here)[https://romm.app/]
